#### Instalacja poleceniem:

npm install (sprawdzić czy bezbłędnie)


#### Start poleceniem:

w dwóch konsolach:
npm run watchCss	- śledzi i kompiluje zmiany w css
npm run watchJs		- śledzi i kompiluje zmiany w js

http://localhost:8080/


#### Build produkcyjnie:

NODE_ENV=production

npm run buildCss
npm run buildJs
